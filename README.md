# NetFlow — Networking Fundamentals for Penetration Testers 🌐

**🔴 Live Site:** [https://rajrathod07.github.io/networking/](https://rajrathod07.github.io/networking/)

An interactive, fluid web guide designed to break down the core networking concepts essential for cybersecurity and penetration testing. 

Before you can exploit a network, you have to understand how it communicates. Built with a modern glassmorphism interface, this project serves as a dynamic study reference for foundational networking rules, topologies, and protocols—critical knowledge for daily pentesting workflows.

## 🌱 A Living Notebook

This project is a continuous work in progress. I am actively studying cybersecurity, and as I learn new networking concepts, protocols, and vulnerabilities, I update this repository. It serves as my personal, interactive textbook that scales up right alongside my skills.

## ✨ Why This Project Exists

In cybersecurity, "enumeration is key," and enumeration relies heavily on networking. This interactive guide was built to solidify the building blocks of network infrastructure:
* **Node Mapping:** Understanding how end devices and intermediaries (routers, switches) interact to identify potential attack vectors.
* **Data Flow:** Grasping full-duplex vs. half-duplex communication to understand packet capture and network sniffing environments.
* **Protocol Rules:** Breaking down encapsulation, encoding, and broadcasting methods to better understand network traffic manipulation.

## 🛠️ Features

* **Modern Glassmorphism UI:** Frosted glass cards and animated background blobs for a distraction-free study environment.
* **Interactive Progress Tracking:** Mark concepts as "done" to track your learning progress (saves locally to your browser).
* **Live Search:** Instantly filter chapters and highlight keywords (e.g., "multicast", "router") as you type.
* **Dark / Light Mode:** Fully themed environments that automatically respect system preferences or manual toggles.
* **Knowledge Checks:** Built-in interactive Q&A sections to test retention on the fly.
* **Fully Responsive:** Adapts seamlessly from desktop monitors to mobile screens for studying on the go.

## 💻 Tech Stack

Built entirely with pure, vanilla web technologies for a lightweight, fast, and secure footprint:
* **HTML5:** Semantic structure and accessible details/summary accordions.
* **CSS3:** Custom properties, Flexbox, CSS Grid, and advanced backdrop-filters.
* **Vanilla JavaScript (ES6+):** DOM manipulation, Intersection Observers, and `localStorage` API.
